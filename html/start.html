Who am I

Hi, my name is Matthew Roybal. I started learning how to program web pages a few years ago on my own, but I wasn't getting very far on my own. Eventually, I joined a Bootcamp and began to hunt for a job. One of the things I noticed missing was the ability to practice job interview questions. So, I started Quizard.me

What is Quizard?

Quizard is a web application that I created to ask those who join questions that may get asked in an interview. Or, simply questions that you should try to get to know the answers. However, my knowledge goes as far as what I've learned so far. The question categories are as follows:

  • HTML
  • CSS
  • SASS
  • JavaScript
  • jQuery
  • NodeJS
  • React
  • Git/Github
  • The Command Line

Since I cannot think of every question myself, I'm hoping you can help me out. I created a form for contributing questions. Just create an account, login and submit your question now!

Why did I start Quizard?

I started out designing Quizard as a project that could show prospective employers what I could do as well as help me prepare better for job interviews. By practing questions and making up more to ask myself over, I figured it was the surest way to get ready for inteviews. If I went to an interview, I would enter all the questiosn afterwards to continue contributing to my own success!

What to expect!

The questions are multiple choice, true and false, and are otherwise pretty basic in format. I hope to get more advanced and be able to include code snippets.
